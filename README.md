## Features

- CLI support in Node.js and Python

## Quick Start

### Node.js

```bash
npm install bhmarketer-crypto-pr
npx crypto-pr-calc 10 global
```

### Python

```bash
pip install bhmarketer-crypto-pr
python -m calculator 10 global
```

```

Estimated Reach: 18,150
```
